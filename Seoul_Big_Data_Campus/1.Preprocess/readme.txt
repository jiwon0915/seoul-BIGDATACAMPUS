재현되지 않는점 양해부탁드립니다.

map_haengjeongdong : 국가공간정보포털에서 가져온 '(센서스경계)행정동경계'데이터를 통해 서울시 행정동 경계지도를 만드는 과정

wifi_preprocess, wifi_preprocess2, wifi_preprocess3
 : 서울시 열린데이터광장에서 가져온 '서울시 공공와이파이 위치정보' 데이터를 가져와서, 
와이파이 주소와 위경도를 바탕으로 행정동을 추출하는 과정

average_income_preprocess : 빅데이터 캠퍼스에서 가져온 '서울시 행정동 단위 생활금융 통계'를 통해 '행정동별 평균 소득'을 구하는 전처리 과정.
이미 행정동별 평규소득을 스케일링해서 가져왔기 때문에, 행정동 코드를 조인하는 과정만 담겨있다.

living_pop : 서울시 열린데이터광장에서 가져온 '행정동별 서울생활인구(내국인)'데이터를 바탕으로 행정동별 평균생활인구를 추출하는 과정

informatization_score : 서울시 열린데이터광장에서 가져온 '서울시 장애인 현황 (등급별/동별) 통계', 
'서울시 국민기초생활보장 수급자 (동별) 통계', '서울시 주민등록인구 (동별) 통계' 을 가져와서 '정보화점수'를 만드는 과정.

buildiing_preprocess : 서울시 빅데이터 캠퍼스 데이터 '028. 서울시 도로명주소 기반 건물 공간데이터/2. 파일데이터/서울시/2016년상'를 활용해서,
shapefile을 변환해서 건물들의 위경도만 추출해 csv로 변환하는 과정이다.
